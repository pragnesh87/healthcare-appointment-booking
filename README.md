## Healthcare Appointment Booking API

Clone Repository

```
git clone git@github.com:pragnesh87/healthcare-appointment-booking.git
```

-   rename .env.example to .env

install dependencies

```
composer install
```

Run command

```
php artisan key:generate
```

Run Migration

```
php artisan migrate --seed
```

Run Application

```
php artisan serve
```
